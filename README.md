# VueCnodejs

> 一个Vue的**初/中级**练手项目：重构CnodeJS社区

### 功能
- [x] 首页列表，下划加载
- [x] 主题详情/评论信息
- [x] 作者信息

### 使用项目
```
1.克隆项目：      git clone https://github.com/geekape/vue-cnode.git
2.安装nodejs
3.安装依赖：      npm install
4.启动服务：      npm run dev
5.发布代码：      npm run build
```

### 技术栈：
---
[Vue.js](https://cn.vuejs.org/)

[Vue-router](https://router.vuejs.org/zh-cn/)
